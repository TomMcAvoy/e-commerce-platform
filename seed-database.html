⚠️ Important

Before running the seeder, make sure MongoDB is running on your system.

If you haven't started MongoDB yet, please do so before continuing:

  • macOS: brew services start mongodb-community
  • Linux: sudo systemctl start mongod
  • Windows: Start MongoDB service from Services
